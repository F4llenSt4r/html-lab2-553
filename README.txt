	จากการทำงานแลปครั้งนี้ ได้เรียนรู้อะไรหลายๆอย่างเกี่ยวกับการเขียนเว็บไซต์ด้วย html และทำให้จากที่เคยสงสัยว่าทำเว็บไซต์ด้วยการเขียนโปรแกรมได้อย่างไร เนื่องจากในระดับมัธยมได้ใช้ google sites ในการทำเว็บไซต์ ตอนนี้ได้เข้าใจถึงวิธีการทำงานและโครงสร้างเบื้องต้นของตัวระบบเว็บไซต์ และเข้าใจความหมายของแต่ละ tags รูปแบบของ elements การทำงานของแต่ละ attributes รวมถึงทราบว่า <style>, style จะอยู่ในรูปแบบของ CSS เนื่องจากเป็นตัวช่วยในการจัดรูปแบบของหน้าเว็บไซต์ที่เกี่ยวข้องกับ สี, รูปแบบฟอนต์ตัวอักษร หรือ การจัดตำแหน่งของ elements ต่างๆ ต่อจากนี้ก็พร้อมที่จะเรียนรู้/ทำงานเกี่ยวกับหัวข้อต่อไปของการเรียน
